# DevOps Engineer Portfolio Deployment with Docker, Kubernetes, and Jenkins

A modern, responsive portfolio website for Ajay Kumar, showcasing DevOps expertise and projects.

## Features

- **Responsive Design** - Works on desktop, tablet, and mobile
- **DevOps Focus** - Tailored content for DevOps/Cloud Engineering roles
- **Contact Form** - Interactive contact form with validation
- **Modern UI** - Clean design with animations and hover effects
- **Separated Code** - HTML, CSS, and JavaScript in separate files
